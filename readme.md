Some missions rely on components from tsp_core which can be found in the [Core Repository](https://github.com/TheSpeshalPlatoon/A3-Core).

Most of the missions in this repository are meant for internal use by TSP and likely require a bunch of mods but some are available on the Steam workshop:

<details>
  <summary><img src="https://core.thespeshalplatoon.com/img/steam_small.png"> <a href="https://steamcommunity.com/sharedfiles/filedetails/?id=3283612524">Killhouse Generator - Alaska</a></summary>

  Welcome to the NATO Reconnaissance Special Warfare Training Center, located somewhere in the remote Pacific North West.
  - 2 Random killhouse generators  
  ↳ 4 themes - Altis, Tanoa, Livonia, Chernarus (RHSGREF), Takistan (3CB Factions)  
  ↳ Configurable locking, furniture, etc  
  ↳ Procedurally generated   
  - 2 Courses of fire   
  - Adjustable lighting, weather, time and music 
  - Skeet shooting
  - Static fast rope (Requires ACE)  
  
  ✔️ If loaded with [RHSGREF](https://steamcommunity.com/workshop/filedetails/?id=843593391), there will be a Chernarus theme option.  
  ✔️ If loaded with [3CB Factions](https://steamcommunity.com/workshop/filedetails/?id=1673456286), there will be a Takistan theme option.  
  ✔️ Compatible with [ACE](https://steamcommunity.com/workshop/filedetails/?id=463939057) fast roping, medical and arsenals.
